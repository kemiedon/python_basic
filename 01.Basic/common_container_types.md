# Python 常用容器資料型別

這份文件將簡要介紹 Python 中四種最常用的容器資料型別：串列（List）、元組（Tuple）、字典（Dictionary）和集合（Set）。

---

## 為什麼需要「容器」？

在寫程式時，我們常會需要儲存「多個資料」。如果我們只用單一變數來儲存，例如：

```python
score1 = 90
score2 = 85
score3 = 78
```

這樣會很難一次處理或統計所有分數。這時候就需要能「裝多個資料」的資料結構，我們稱之為 **資料容器（Data Container）**。

## 🧺 什麼是資料容器？

資料容器就像不同類型的「資料盒子」，每種盒子有不同的特性和用途。

| 類型              | 說明                  | 範例                                                   | 特性                     |
| :---------------- | :-------------------- | :----------------------------------------------------- | :----------------------- |
| **List（串列）**  | 有順序、可修改        | `["apple", "banana", "cherry"]`                        | 可新增、刪除、改值       |
| **Tuple（元組）** | 有順序、不可修改      | `("red", "green", "blue")`                             | 用於保護資料不被意外修改 |
| **Dict（字典）**  | 鍵值配對（key:value） | `{"name": "Tom", "age": 18}`                           | 可快速查找資料           |
| **Set（集合）**   | 無順序、不重複        | `{"apple", "banana", "apple"}` → `{"apple", "banana"}` | 自動去除重複值           |

## 📦 容器的概念圖（口語比喻）

如果用生活中的例子來比喻，可以這樣理解：

| Python 容器 | 比喻                                         |
| :---------- | :------------------------------------------- |
| `list`      | 一排有編號的儲物櫃，可以隨時換物品           |
| `tuple`     | 一排被封死的透明信箱，只能看不能動           |
| `dict`      | 貼著不同標籤的抽屜（如「帳單」、「情書」等） |
| `set`       | 把重複的信件自動丟掉的篩選器                 |

## 切片 (Slicing)

切片 (Slicing) 是一種強大的功能，可以讓我們從序列（如字串、串列）中取出「一部分」。它的語法非常直觀，不僅適用於字串，也適用於 `list` 和 `tuple`。

**語法：** `sequence[start:stop:step]`

- `start`：起始索引（包含），如果省略，預設為 0。
- `stop`：結束索引（**不包含**），如果省略，預設為序列的結尾。
- `step`：間隔值（步長），如果省略，預設為 1。

### 字串切片範例

讓我們用一個字串 `s = "Hello, Python!"` 來當作範例：

```python
s = "Hello, Python!"

# 範例 1: 基本切片 (從索引 7 取到索引 12)
substring1 = s[7:13]
print(f"從索引 7 到 13: '{substring1}'")  # 結果: 'Python'

# 範例 2: 從頭開始切片 (省略 start)
substring2 = s[:5]
print(f"從頭到索引 5: '{substring2}'")    # 結果: 'Hello'

# 範例 3: 切片到結尾 (省略 stop)
substring3 = s[7:]
print(f"從索引 7 到結尾: '{substring3}'")  # 結果: 'Python!'

# 範例 4: 使用負數索引 (從倒數第 7 個取到倒數第 2 個)
substring4 = s[-7:-1]
print(f"使用負數索引: '{substring4}'")      # 結果: 'Python'

# 範例 5: 使用 step 間隔取值 (每隔 2 個字元取一個)
every_other = s[::2]
print(f"間隔取值: '{every_other}'")      # 結果: 'Hlo yhn'

# 範例 6: 反轉字串 (一個常見的技巧是使用 step = -1)
reversed_s = s[::-1]
print(f"反轉字串: '{reversed_s}'")        # 結果: '!nohtyP ,olleH'
```
